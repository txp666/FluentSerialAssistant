# 安全策略

## 支持范围

当前项目处于早期开发阶段，默认只维护主分支上的最新代码。正式发布版本后，可以在本文件中补充具体受支持版本。

## 报告安全问题

如果你发现安全问题，请不要在公开 Issue 中直接披露可利用细节。

建议报告内容包含：

- 影响版本或提交
- 影响平台
- 复现步骤
- 潜在影响
- 可行的缓解方式

在没有专用安全邮箱前，请先通过仓库维护者提供的私有渠道联系。确认问题后再公开修复说明。

## 项目安全边界

Fluent 串口助手会与本机串口设备通信，并可能读写导出的会话文件。请注意：

- 不要打开不可信来源提供的配置或脚本。
- 不要把包含密钥、令牌、设备凭据的串口日志提交到公开 Issue。
- 导出的 TXT、CSV、BIN 可能包含敏感设备数据，分享前请自行脱敏。

## 依赖安全

项目依赖 Qt 和 FluentQtWidgets。请定期关注：

- Qt 官方安全公告
- FluentQtWidgets 上游更新
- GitHub Actions 依赖 Action 的版本更新
