# Fluent 串口助手 v0.1.5

这是一个终端显示增强和代码结构整理版本。

## 新增

- 终端时间戳使用独立颜色显示。
- RX/TX 方向符号使用独立颜色显示。
- ESP-IDF `V/D/I/W/E` 日志等级前缀支持分色显示，例如 `I (4929)`、`W (3639)`、`E (3639)`。
- 增加项目级 `.clang-format`、格式化脚本和 CMake `format`、`format-check` 目标。

## 变更

- 将 Workbench 页面拆分为多个职责清晰的实现文件，提升后续维护可读性。
- 全量格式化项目 C/C++ 源码。

## 验证

- `cmake --build build/codex-release --target format-check`
- `cmake --build build/codex-release --parallel`
