# Fluent 串口助手 v0.1.6

这是一个协议调试能力和左侧布局体验增强版本。

## 新增

- 接收显示和文本发送支持分别选择 UTF-8、GBK、ASCII、Latin1 编码。
- 增加校验工具，支持 CRC16-Modbus、CRC16-CCITT、CRC32、LRC、XOR、SUM8 计算。
- 支持校验结果大小端显示，并可在发送时自动追加校验码。
- 自动断帧支持按接收间隔、固定帧头、固定帧尾和固定长度断帧。
- 增加 Modbus RTU 辅助，可生成常用读写请求帧、自动追加 CRC、填入发送区或直接发送。
- Modbus RTU 响应支持基础摘要解析，包括站号、功能码、数据、异常码和 CRC 状态。
- 左侧功能卡片支持折叠，并记住每张卡片的折叠状态。

## 修复

- 修复常用包和文件发送卡片折叠后仍残留内容区空白高度的问题。

## 验证

- `cmake --build build/codex-release --target format-check`
- `cmake --build build/codex-release --parallel`
