# Fluent 串口助手 v0.1.3

这是一个功能增强和 macOS 窗口显示修复版本。

## 新增

- 终端支持关键字搜索和全部、接收、发送方向过滤。
- 终端标题栏显示 RX/TX 总量、实时速率和连接时长。
- 支持保存、载入、发送、删除和排序常用发送包。
- 支持选择文件后按块发送，并配置块大小、块间隔和查看发送进度。
- 支持恢复上次发送输入、常用包、文件发送参数，并可选启动后自动连接上次串口。

## 修复

- 修复 macOS 无边框 Fluent 窗口底部圆角未裁剪导致内容露出的问题。
- macOS 窗口圆角裁剪改为原生 AppKit layer 实现，避免 Qt mask 圆角锯齿。

## 发布资产

- Windows x64：包含 `windeployqt` 部署后的运行文件。
- macOS：包含 `.app` 应用包。
- Linux x64：包含可执行文件和项目文档，系统运行库需由发行版提供。
