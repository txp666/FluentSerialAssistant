# Fluent 串口助手 v0.1.9

这是一个规划收口和专业调试能力补齐版本。前期规划中的数据表格、脚本插件和协议模板已经完成；串口转发 / 桥接不再纳入当前产品范围。

## 新增

- 增加数据表格视图，可按帧查看时间、方向、来源、长度、HEX、文本和校验结果，支持排序、过滤、复制单帧/HEX，并可定位回终端记录。
- 增加脚本插件能力，支持 JavaScript 脚本载入、保存、示例、运行、停止和日志输出。
- 脚本插件提供受控 `serial` API，可发送文本/HEX、读取收发记录快照和写脚本日志。
- 增加协议模板能力，可定义帧头、长度字段、命令字、载荷和尾部校验字段，并按当前模板解析接收帧。
- 协议模板示例改为详细说明弹窗，包含示例帧、字段位置、对应配置和长度字段用法。

## 变更

- 业务配置固定写入安装目录下的 `FluentSerialAssistant.ini`，不再使用 Windows 注册表或用户配置目录。
- Fluent 外观配置固定写入安装目录下的 `FluentSerialAssistant.fluent.json`。
- 左侧卡片默认仅连接卡片展开，其余卡片默认收起，并继续记住用户折叠状态。
- 数值配置去除 SpinBox，改为紧凑数字文本输入，减少侧栏拥挤和显示截断。
- 快捷按钮提示、协议示例说明和字体导入失败提示统一使用 Fluent 组件库。
- 串口转发 / 桥接不纳入当前产品计划，已移除功能路线图文件。

## 修复

- 修复协议模板命令和载荷字段在侧栏窄宽度下显示不全的问题。
- 修复协议模板示例提示过短、无法说明字段取数方式的问题。
