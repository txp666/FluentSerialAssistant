# Fluent 串口助手 v0.1.4

这是一个发布包修复版本。

## 修复

- macOS 发布包写入应用 `.icns` 图标，Finder 和 Dock 会显示应用 logo。
- Linux 发布包不再只包含裸可执行文件，改为携带 Qt 运行库、Qt 插件、启动脚本和桌面图标元数据。
- 增加 GitHub Social preview 预览图，可上传到仓库 Settings 以替换默认动态卡片图片。

## 发布资产

- Windows x64：包含 `windeployqt` 部署后的运行文件。
- macOS：包含 `.app` 应用包和应用图标。
- Linux x64：包含可执行启动脚本、Qt 运行库、Qt 插件和桌面图标元数据。
