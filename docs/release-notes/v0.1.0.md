# Fluent 串口助手 v0.1.0

首个公开预览版本，聚焦串口终端工作台和基础设置能力。

## 主要功能

- Fluent 风格串口终端工作台。
- 串口扫描、连接、断开和自动重连。
- 波特率、数据位、校验位、停止位、流控、RTS、DTR 配置。
- 文本、HEX、混合三种终端显示模式。
- 自动断帧显示和时间戳。
- TX 记录颜色选择。
- 文本和 HEX 数据发送。
- None、CR、LF、CRLF 行结束符。
- 发送历史和循环发送。
- RX/TX 字节计数。
- 暂停显示、自动滚动、清空终端、重置计数。
- TXT、CSV、BIN 会话导出。
- 接收数据保存到文件。
- 浅色、深色、跟随系统主题和主题色配置。

## 发布资产

- Windows x64：包含 `windeployqt` 部署后的运行文件。
- macOS：包含 `.app` 应用包。
- Linux x64：包含可执行文件和项目文档，系统运行库需由发行版提供。

## 已知限制

- 暂未提供自动化 GUI 测试。
- 暂未开放包列表、快速绘图、宏命令和测试序列。
