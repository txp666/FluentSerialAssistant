# 变更记录

本项目遵循面向用户的变更记录格式。版本号在正式发布前可能继续调整。

## [0.1.0] - 未发布

### 新增

- Fluent 风格串口终端工作台。
- 串口扫描、连接和断开。
- 波特率、数据位、校验位、停止位、流控、RTS、DTR 配置。
- 文本、HEX、混合三种终端显示模式。
- 自动断帧显示。
- TX 记录颜色选择。
- 文本和 HEX 数据发送。
- None、CR、LF、CRLF 行结束符。
- 发送历史记录。
- 循环发送。
- RX/TX 字节计数。
- 暂停显示、自动滚动、清空终端、重置计数。
- TXT、CSV、BIN 会话导出。
- 接收数据保存到文件。
- 浅色、深色、跟随系统主题和主题色配置。
- Windows 紧凑发布包脚本。
- GitHub Actions 三平台构建 CI。

### 变更

- UI 聚焦终端和设置两页，隐藏未完成入口。
- 主窗口默认尺寸调整为 `1120x900` 并启动居中。
- 发送按钮改为纯图标按钮。

### 已知限制

- 暂未提供自动化 GUI 测试。
- 暂未提供 macOS 和 Linux 打包脚本。
- 暂未开放包列表、快速绘图、宏命令和测试序列。
