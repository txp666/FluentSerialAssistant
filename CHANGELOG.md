# 变更记录

本项目遵循面向用户的变更记录格式。版本号在正式发布前可能继续调整。

## [0.1.9] - 2026-07-09

### 新增

- 增加数据表格视图，可按帧查看时间、方向、来源、长度、HEX、文本和校验结果，支持排序、过滤、复制单帧/HEX，并可定位回终端记录。
- 增加脚本插件能力，支持 JavaScript 脚本载入、保存、示例、运行、停止和日志输出，可通过受控 `serial` API 发送文本/HEX、读取收发记录快照和写脚本日志。
- 增加协议模板能力，支持定义帧头、长度字段、命令字、载荷和尾部校验字段，模板可保存、删除、切换和启用，并可按模板解析接收帧。
- 协议模板示例改为详细说明弹窗，包含示例帧、字段位置、对应配置和长度字段用法。

### 变更

- 业务配置改为固定写入安装目录下的 `FluentSerialAssistant.ini`，不再使用 Windows 注册表或用户配置目录；Fluent 外观配置写入同目录的 `FluentSerialAssistant.fluent.json`。
- 左侧卡片默认仅连接卡片展开，其余卡片默认收起，并继续记住用户折叠状态。
- 协议模板、日志、帧设置等数值输入去除 SpinBox，改为紧凑数字文本输入，避免侧栏宽度不足。
- 快捷按钮提示、协议示例说明和字体导入失败提示统一使用 Fluent 组件库。
- 串口转发 / 桥接不纳入当前产品计划，已移除功能路线图文件。

### 修复

- 修复协议模板命令和载荷字段在侧栏窄宽度下显示不全的问题。
- 修复协议模板示例提示过短、无法说明字段取数方式的问题。

## [0.1.8] - 2026-07-09

### 新增

- 自动日志增强为 TXT、CSV、BIN 三种格式，可按单文件大小滚动，日志文件名包含时间、串口名和波特率，并在侧栏显示当前写入状态。
- 增加多标签会话，标签放在标题栏顶部，每个会话拥有独立串口、终端、发送区、统计和日志配置，新标签会复制当前会话配置。
- 增加快速绘图独立窗口，可从接收文本按全部数字、分隔值、键值对和 JSON 对象协议取数，支持多通道实时曲线、协议示例提示、暂停、清空、复位视图和 CSV 导出。
- 增加应用中英文翻译资源和语言设置，支持简体中文、English、跟随系统，并可在标题栏快捷按钮中无需重启切换。
- 增加标题栏快捷操作，搜索、快速绘图、语言、接收区 HEX/文本、主题和设置入口集中放在统计信息右侧。

### 变更

- 终端搜索改为弹出悬浮窗口，减少终端标题区域占用。
- 终端统计信息靠左、快捷图标靠右，标题栏标签和 logo 间距更稳定。
- 发送区增加 HEX/文本快捷切换，接收区增加 HEX/文本图标切换。
- 侧栏表单英文布局改为统一标签列宽，并压缩部分英文短文案，避免切换英文后控件显示不全或上下不对齐。

### 修复

- 修复标题栏标签在新增标签、切换主题或切换标签后高度、底部强调色和显示区域回退的问题。
- 修复标题栏 logo 在多标签会话布局中消失的问题。
- 修复新增快捷按钮垂直不居中和文本按钮在标题栏高度内显示不稳的问题。
- 快速绘图曲线右键菜单改为中文显示。

## [0.1.7] - 2026-07-09

### 新增

- 终端搜索支持保留原记录并高亮命中内容，可上/下跳转匹配，支持大小写敏感、正则搜索和方向过滤。
- 常用包支持分组、备注、启用/停用、批量发送、排序和 JSON 导入导出。
- 增加宏命令能力，支持多步骤发送、步骤间延时、等待文本/HEX 响应、循环运行、失败中止和 CSV 结果导出。
- 增加自动应答能力，支持文本、HEX、正则匹配，命中后可立即或延时发送文本/HEX 应答，并在发送记录中标记来源。

### 变更

- 更新检查信息条调整到顶部显示，避免遮挡主要操作区域。

### 修复

- 修复左侧卡片内部控件获得鼠标滚轮时，侧边面板无法继续滚动的问题。

## [0.1.6] - 2026-07-08

### 新增

- 接收显示和文本发送支持分别选择 UTF-8、GBK、ASCII、Latin1 编码，并持久化保存配置。
- 增加校验工具，支持 CRC16-Modbus、CRC16-CCITT、CRC32、LRC、XOR、SUM8 手动计算、大小端显示和发送时自动追加。
- 自动断帧增强为支持按接收间隔、固定帧头、固定帧尾和固定长度断帧。
- 增加 Modbus RTU 辅助，支持常用读写请求帧生成、CRC16-Modbus 自动追加、填入发送区、直接发送和响应摘要解析。
- 左侧连接、接收设置、发送设置、Modbus RTU、常用包和文件发送卡片支持折叠，并记住每张卡片的折叠状态。

### 修复

- 修复常用包和文件发送卡片折叠后仍残留内容区空白高度的问题。

## [0.1.5] - 2026-07-08

### 新增

- 终端支持时间戳、RX/TX 方向符号和 ESP-IDF `V/D/I/W/E` 日志等级分色显示，并适配深色和浅色主题。
- 增加项目级 `.clang-format`、格式化脚本和 CMake `format`、`format-check` 目标。

### 变更

- 将 Workbench 页面按布局、左侧设置区、终端渲染、常用包、文件发送、连接控制和状态管理拆分到 `src/app/view/workbench/`，降低单文件复杂度。
- 使用 clang-format 统一格式化项目 C/C++ 源码。

## [0.1.4] - 2026-07-08

### 修复

- macOS 发布包写入应用 `.icns` 图标，修复 Finder 和 Dock 中不显示应用图标的问题。
- Linux 发布包改为携带 Qt 运行库、Qt 插件、启动脚本和桌面图标元数据，避免只发布裸可执行文件导致包体异常小且依赖系统 Qt。
- 增加 GitHub Social preview 预览图，用于替换默认仓库动态卡片中的 owner 头像。

## [0.1.3] - 2026-07-08

### 新增

- 终端支持关键字搜索和全部、接收、发送方向过滤。
- 终端标题栏显示 RX/TX 总量、实时速率和连接时长。
- 支持保存、载入、发送、删除和排序常用发送包。
- 支持选择文件后按块发送，并配置块大小、块间隔和查看发送进度。

## [0.1.0] - 2026-07-07

### 新增

- Fluent 风格串口终端工作台。
- 串口扫描、连接和断开。
- 波特率、数据位、校验位、停止位、流控、RTS、DTR 配置。
- 文本、HEX、混合三种终端显示模式。
- 自动断帧显示。
- TX 记录颜色选择。
- 文本和 HEX 数据发送。
- None、CR、LF、CRLF 行结束符。
- 发送历史记录。
- 循环发送。
- RX/TX 字节计数。
- 暂停显示、自动滚动、清空终端、重置计数。
- TXT、CSV、BIN 会话导出。
- 接收数据保存到文件。
- 浅色、深色、跟随系统主题和主题色配置。
- Windows 紧凑发布包脚本。
- GitHub Actions 三平台构建 CI。
- GitHub Releases 三平台发布资产。

- 主窗口默认尺寸调整为 `1120x900` 并启动居中。
- 发送按钮改为纯图标按钮。
- 接收设置和发送设置的双列复选框对齐。
